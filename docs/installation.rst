Installation Guide
==================

.. toctree::
  :maxdepth: 2


PyPi
----

Install the Hydrobox using pip. The latest version on PyPI_ can
be installed using pip:

.. _PyPI: https://pypi.python.org/pypi/hydrobox

.. code-block:: bash

    pip install hydrobox


GitHub
------

There might be a more recent version on GitHub available. This can be installed like:

.. code-block:: bash

    git clone https://git.scc.kit.edu/aw6481/hydrobox.git
    cd hydrobox
    pip install -r requirements.txt
    pip install -e .
